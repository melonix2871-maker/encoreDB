📡 Full REST API Endpoints

All endpoints accept POST requests with JSON body. Returns JSON responses.

GET /api/records.html - Get all records
Supports filters: ?collection=user&owner_id=1
GET /api/record.html?id=5 - Get single record
POST /api/create.html - Create new record
Body: {owner_id, relational_id, repeater, collection, meta_key, meta_value, access_key}
✓ Validates: no duplicate (owner_id + collection + meta_key must be unique)
POST /api/update.html - Update record
Body: {id, ...fields to update}
✓ Validates: no duplicate conflicts on update
POST /api/delete.html - Delete record
Body: {id}
🔒 Validation Rules:
• Unique constraint: (owner_id + collection + meta_key) combination must be unique
• Owner ID is required
• Meta Key is required
• ID is auto-incremented
• Date Created is auto-generated
